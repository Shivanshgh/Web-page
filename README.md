# Personal Portfolio Website
This is a simple personal portfolio website built using **HTML** and **CSS**. It was created as a beginner web development project to practice building and publishing a website with GitHub Pages.

## Features
- Responsive and clean design
- About Me section
- Projects section
- Contact section
- Smooth scrolling navigation
- Animated envelope on the home page

## Technologies Used
- HTML5
- CSS3

## AI Usage
I designed and built the website myself. The only part where I used AI assistance was for creating the animated envelope on the landing page. Everything else, including the overall layout, content, styling, and project structure, was implemented and customized by me.
